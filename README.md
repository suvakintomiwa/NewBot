# Web3 Jarvis Bot

Elite AI-powered Web3 operations terminal built in Python. Deployed on Railway.

## Features
- Multi-provider AI assistant
- Crypto discovery (trending, meme coins, new launches)
- Project analysis (contract, website)
- Web3 job hunter (CM, mod, dev, KOL)
- Auto alerts
- Watchlist & notes

## Setup
1. Rename `.env.example` to `.env` and add your free API keys.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`

## Railway Deployment
Push to GitHub, connect to Railway, set environment variables, deploy.
